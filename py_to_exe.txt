Comandos para converter de .py para .exe

IMPORTANTE: Corrigir o --icon <location> para o caminho onde estiver guardado o icon, --add-data <folder> para o ficheiro onde estiverem guardadas as imagens que são usadas no programa e o último argumento para a localização do ficheiro main

MUITO IMPORTANTE: Por favor!! não ponham nenhum dos ficheiros já convertidos no github. Não há razão nenhuma para ter lá 2Gb de porcaria.

**SINGLE FILE**

pyinstaller --noconfirm --onefile --windowed --hidden-import altgraph --icon "C:/Users/tsfei/anaconda3/CHIMERA/CHIMERA/img/favicon.ico" --name "CHIMERA" --add-data "C:/Users/tsfei/anaconda3/CHIMERA/CHIMERA/img;img/" --exclude-module PyQt5 "C:/Users/tsfei/anaconda3/CHIMERA/CHIMERA/main.py"

**SINGLE FOLDER**

pyinstaller --noconfirm --onedir --windowed --hidden-import altgraph --icon "C:/Users/tsfei/anaconda3/CHIMERA/CHIMERA/img/favicon.ico" --name "CHIMERA" --add-data "C:/Users/tsfei/anaconda3/CHIMERA/CHIMERA/img;img/" --exclude-module PyQt5 "C:/Users/tsfei/anaconda3/CHIMERA/CHIMERA/main.py"
